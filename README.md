# l1 Norm
Implementation of the [l1 norm](https://en.wikipedia.org/wiki/Norm_(mathematics)#Taxicab_norm_or_Manhattan_norm).  

Compute the Manhattan/Taxicab/l1 norm of a vector.  

```bash
npm install l1-norm
yarn add l1-norm
```

```js
const l1norm = require('l1-norm');
l1norm([64, 42, 21]);
// 127
```
